# KGH1003: Prefer is null

## Summary

When checking for null, use the `is null` pattern instead of the `==` operator.

## Rationale

This rule makes null checks more explicit and avoids the effects of overloaded
equality operators.

## Before

```csharp
if (value == null)
```

```csharp
if (value =! null)
```

## After

```csharp
if (value is null)
```

```csharp
if (value is not null)
```

## Applies to

- `== null`
- `!= null`

## Code fixes

- `PreferIsNull`: Replaces `== null` with `is null`.

## Configuration

None.
