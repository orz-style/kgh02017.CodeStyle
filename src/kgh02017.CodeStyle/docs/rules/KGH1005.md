# KGH1005: Prefer using declaration

## Summary

When using a `using` statement, prefer a `using` declaration instead.

## Rationale

This rule reduces nesting, eliminates unnecessary braces, and improves readability.

## Before

```csharp
using (var stream = new MemoryStream())
{
    DoSomething(stream);
}
```

## After

```csharp
using var stream = new MemoryStream();
DoSomething(stream);
```

## Applies to

- `using` statements with a local variable declaration

## Code fixes

- `PreferUsingDeclaration`: Replaces a `using` statement with a `using` declaration.

## Configuration

None.
