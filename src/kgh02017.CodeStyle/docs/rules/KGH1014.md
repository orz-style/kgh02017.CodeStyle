# KGH1014: Prefer leading continuation operators

## Summary

When an expression spans multiple lines, use a consistent layout by
placing either continuation operators at the beginning of the continued
line or the entire expression on a single line.

## Rationale

This rule improves readability by making the relationship between
operators and operands easier to recognize in multiline expressions.

## Before

```csharp
if (a &&
    b)
{
}
```

## After

```csharp
// Option 1: UseLeadingOperators
if (a
    && b)
{
}

// Option 2: UseSingleLineExpression
if (a && b)
{
}

```

## Applies to

- `&&`
- `||`
- `??`
- `?:`

## Code fixes

- `UseLeadingOperators`: Places continuation operators at the beginning of the continued line.
- `UseSingleLineExpression`: Places the entire expression on a single line.

## Configuration

None.
