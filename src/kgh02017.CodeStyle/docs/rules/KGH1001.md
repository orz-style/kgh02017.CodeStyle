# KGH1001: Do not use interpolated strings in logger calls

## Summary

When using logging APIs, use structured logging instead of interpolated strings.

## Rationale

This rule preserves named log properties, making logs easier to search,
filter, and analyze.

## Before

```csharp
_logger.LogInformation(
    $"Updating spool {shortCode}.");
```

## After

```csharp
_logger.LogInformation(
    "Updating spool {ShortCode}.",
    shortCode);
```

## Applies to

- Microsoft.Extensions.Logging

## Code fixes

None.

None.
