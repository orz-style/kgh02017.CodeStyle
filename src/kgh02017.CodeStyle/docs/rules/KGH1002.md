# KGH1002: Specify StringComparison

## Summary

When calling string comparison APIs, specify the StringComparison argument explicitly.

## Rationale

This rule helps avoid unexpected behavior caused by culture-dependent string comparisons.
For non-linguistic string comparisons, `StringComparison.Ordinal` is generally recommended.

## Before

```csharp
value.StartsWith("abc");
```

## After

```csharp
value.StartsWith("abc", StringComparison.Ordinal);
```

## Applies to

- `string.Equals`
- `string.StartsWith`
- `string.EndsWith`
- `string.Contains`
- `string.Compare`
