# KGH1004: Line exceeds 120 characters

## Summary

When a line exceeds 120 characters, split it across multiple lines.

## Rationale

This rule improves readability and helps maintain consistent formatting.

## Before

```csharp
SyntaxTriviaList argumentLeadingTrivia = SyntaxFactory.TriviaList(SyntaxFactory.EndOfLine(newLine), SyntaxFactory.Whitespace(argumentIndent));
```

## After

```csharp
SyntaxTriviaList argumentLeadingTrivia =
    SyntaxFactory.TriviaList(
        SyntaxFactory.EndOfLine(newLine),
        SyntaxFactory.Whitespace(argumentIndent));
```

## Applies to

- All C# code

## Code fixes

None.

## Configuration

The maximum line length is currently fixed at 120 characters
