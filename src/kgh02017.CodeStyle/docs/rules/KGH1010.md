# KGH1010: Prefer PascalCase logger template names

## Summary

When using structured logging placeholders, use the PascalCase naming convention.

## Rationale

This rule promotes consistent property names, making structured logs easier
to search and analyze.

## Before

```csharp
_logger.LogInformation("Updating spool {shortCode}.", shortCode);
```

## After

```csharp
_logger.LogInformation("Updating spool {ShortCode}.", shortCode);
```

## Applies to

- Microsoft.Extensions.Logging

## Code fixes

None.

## Configuration

None.
